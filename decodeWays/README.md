# Decode ways

A message containing letters from A-Z is being encoded to numbers using the following mapping:

'A' -> 1

'B' -> 2

...

'Z' -> 26

Objective is to determine the total number of ways to decode it.

### Sample I/O:

(1)

Input: "12"

Output: 2

Explanation: "AB" (1 2) or "L" (12)

(2)

Input: "1235226"

Output: 9
